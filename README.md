## iobscan-ibc-explorer
IBC Explorer Frontend


## development
step1 安装依赖
```
 	yarn install
```
step2 服务启动, 后端服务地址可修改.env.development
```
  yarn dev
```

step3 preview
```
	yarn preview
```

## qa
step1 安装依赖
```
 	npm install
```

step2 build

```
  npm run build:qa
```
